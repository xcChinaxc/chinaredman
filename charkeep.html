Amplifying Gaming Sessions with Character Keeper

Character Keeper is a full-stack web application that users use to track character notes for MMOs, tabletop games, and more!

Project Purpose and Goal

I bounce between various games and cannot remember the little details needed to advance between playthroughs. I got tired of wasting those valuable in-game days retracing my steps to jog my memory, so I built this application to amplify my gaming sessions!

Spotlight

I'm currently in the process of refactoring this app with Gatsby.js! So, let's chat about the details of this project, or you can explore the app yourself. Either way, I hope to hear from you!
